## ğŸ¯ Project Overview

### BACK-END
Back-end development learning materials and projects:
- **Express.js**: Web server framework setup and routing
- **EJS**: Server-side templating engine examples
- **Node.js**: Core JavaScript runtime projects including:
  - Math utilities
  - Figlet ASCII art generation
  - Object-oriented programming with fruits

Front-end development learning materials and interactive projects:
- **HTML**: Static pages and web assets

  - Guess Favorite Movie game
  - Random Number Game
  - Loop demonstrations
  - General utilities
- **Styled Projects**: Complete interactive projects with HTML, CSS, and JavaScript

### MISCELLANEOUS
Experimental and additional learning materials for both front-end and back-end development.

## ğŸš€ Getting Started

### Prerequisites
- Node.js and npm installed on your system
- A text editor (VS Code recommended)
- Basic understanding of JavaScript, HTML, and CSS

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ashutoshsharma8204/Sigma-Prime-Batch.git
cd Sigma-Prime-Batch
```

2. Install dependencies for back-end projects:
```bash
# For Express server
cd BACK-END/Express-Dir
npm install

# For Node.js projects
cd ../Node-Js
npm install

# For EJS
cd ../EJSDIR
npm install
```
